Description of key bindings

  • `ENTER` submits a note
  • `s` starts/stops a shift
  • `p` pauses/resumes a shift
  • `w` is a faceoff win, starts a shift
  • `l` is a faceoff win, starts a shift
  • `g` is a goal, plus, ends a shift
  • `a` is an assist, plus, ends a shift
  • `RIGHT` is a plus, ends a shift
  • `LEFT` is a minus, ends a shift
  • `x` is a shot
  • `b` is a blocked shot
  • `o` is an off-target shot
  • `h` is a hit
  • `r` is reload and new game

    • Hockey Stats App v0.1.0, October 2019.
